# 다시이음 (WorkLinker)

중장년층의 재취업 준비를 돕기 위해 정부지원 및 무료 교육 정보를 한곳에서 탐색하고, 사용자 조건에 맞는 교육을 추천받을 수 있도록 기획한 중장년 맞춤형 교육 추천 서비스입니다.

## 프로젝트 소개

재취업을 준비하는 중장년층은 교육 정보, 지원 제도, 직무 정보가 여러 사이트에 분산되어 있어 필요한 정보를 한 번에 비교하기 어렵습니다.

다시이음은 이러한 문제를 해결하기 위해 정부지원 및 무료 교육 정보를 통합 제공하고, 사용자 관심 분야와 조건에 맞는 교육을 추천하여 재취업 준비 과정을 지원하는 것을 목표로 했습니다.

## 주요 기능
- 정부지원 및 무료 교육 프로그램 탐색
- 사용자 관심 분야 기반 교육 추천
- 지역·연령·성별 기반 취업 동향 정보 제공
- 맞춤 직무 추천
- 성공 사례 게시판
- 로그인 및 회원 가입
- 게시글 및 댓글 기능
- 일정 관리 스케쥴
- AI 챗봇

## 기획 및 설계 역량
이 프로젝트에서는 단순 화면 구현보다 중장년 재취업 준비 과정에서 필요한 정보 탐색 흐름을 설계하는 것에 집중했습니다.

- 중장년 재취업을 위한 무료 교육 정보 접근성 어려움에 대한 문제 정의
- 교육 탐색 -> 교육 추천 -> 성공 사례 -> 취업동향 -> 직무 추천으로 이어지는 서비스 흐름 설계
- 중장년층 사용자를 고려한 큰 글씨, 단순한 메뉴 구조, 카드형 정보 배치 적용
- 사용자 조건 기반 정보 탐색 구조 설계
- 취업동향 데이터를 서비스 화면에 연결하는 대시보드 구조 기획
- 주요 화면의 정보 우선순위 및 UI 구성 설게

## 데이터 출처 및 활용
본 프로젝트는 중장년층 재취업 준비 과정에서 필요한 교육·취업 정보를 제공하기 위해 공공 데이터를 기반으로 서비스 구조를 설계했습니다. 

### 활용 데이터
- 고용 24 / HRD-Net 교육 데이터
 - 직업 훈련 과정명 및 교육 정보를 기반으로 교육 추천 화면 구성
 - 훈련 과정명을 기준으로 분야, 기술, 도구, 업무, 메타 키워드로 구조화
- 고용행정통계 CSV 데이터
 - 지역별 취업건수
 - 연령별 취업건수
 - 성별 취업건수
 - 직무별 취업동향 데이터
- NCS 직무 체계
 - 직무 분류 및 추천 직무 구조 설계 참고
 - 교육 과정과 직무 수요를 연결하기 위한 기준 데이터로 활용
- 서을시 50플러스재단/ 서울기술교육원 교육 정보
 - 중장년 대상 교육 프로그램 구성 및 무료 직업 훈련 과정 구조 참고
 - 중장년층 대상 서비스 흐름과 교육 정보 분류 방식 참고

### 데이터 가공 방식
 - 수집한 교육 과정명을 기준으로 핵심 키워드를 추출
 - 교육 데이터를 분야, 기술, 도구, 업무, 메타 기준으로 분류
 - 사용자 관심 분야와 교육 키워드를 비교하여 추천 흐름 구성
 - 취업동향 CSV 데이터를 지역·연령·성별 기준으로 정리하여 대시보드 화면에 반영
 - 추천 직무 TOP5화면을 통해 사용자 조건에 맞는 직무 탐색 흐름 제공

### 향후 데이터 고도화 방향
 - HRD-Net 및 공공 교육 API 연동 구조 확대
 - 서울시 50플러스재단, 서울기술교육원 등 중장년 특화 교육 데이터 연계
 - 교육 과정명 외 커리큘럼, 교육 내용, 대상 조건 등 세부 데이터 확보
 - 공공 API 기반 자동 업데이트 배치 구축
 - 교육 데이터와 채용 데이터를 연결한 직무 추천 정확도 개선

## 협업 경험

프로젝트 진행 중 백엔드 담당 인원이 이탈하면서 일부 DB 구조 설계와 백엔드 연동 작업을 함께 수행했습니다.
이 과정에서 화면 구현 뿐 아니라 데이터 구조와 API 응답 방식이 실제 사용자 경험에 직접적인 영향을 준다는 점을 경험했습니다. 
PM의 기획 방향 안에서 팀원으로서 화면 구현, 사용자 흐름 개선, 데이터 연결 보완에 기여 했습니다.
